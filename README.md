## Overview

Comprehensive NLP pipeline featuring sentiment analysis and Named Entity Recognition (NER) using BERT and Hugging Face Transformers. Deployed at **Mizuho Bank** for analyzing customer feedback and financial text data.

## Key Features

- **Sentiment Analysis** using BERT for customer feedback and social media interactions
- **Named Entity Recognition (NER)** with Hugging Face Transformers and spaCy
- **Product name, size, and preference extraction** from customer reviews
- **Abstractive Summarization** using Llama2 and LangChain
- **Real-time Intent Recognition** for customer support chatbots (20% accuracy improvement)
- **Interactive Streamlit Dashboard** for marketing and operations teams
- **NLP Pipelines** integrated with Flask REST APIs

## Use Cases

1. **Financial Text Analysis** - Extract insights from banking transaction comments
2. **Customer Support** - Intent recognition for chatbot query resolution
3. **Product Analytics** - Extract product attributes from customer reviews
4. **Social Media Monitoring** - Real-time sentiment tracking

## Results

- 20% improvement in customer support query resolution accuracy
- Actionable product insights provided to development teams
- Scalable NLP inference on Azure cloud with low latency
